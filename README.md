# error-html

一组精美的 HTTP 错误页面模板，采用统一的设计风格，适用于各种 Web 项目。

## 包含的错误页面

| 状态码 | 页面文件 | 说明 |
|--------|----------|------|
| 400 | [400_错误请求.html](400_错误请求.html) | 请求参数错误或格式不正确 |
| 401 | [401_未授权.html](401_未授权.html) | 需要提供身份验证凭据 |
| 403 | [403_禁止访问.html](403_禁止访问.html) | 没有权限访问此资源 |
| 404 | [404_资源未找到.html](404_资源未找到.html) | 请求的资源不存在或已被删除 |
| 500 | [500_服务器错误.html](500_服务器错误.html) | 服务器内部错误 |
| 502 | [502_错误网关.html](502_错误网关.html) | 网关或代理收到无效响应 |
| 503 | [503_服务不可用.html](503_服务不可用.html) | 服务器暂时无法处理请求 |
| 504 | [504_网关超时.html](504_网关超时.html) | 网关或代理未能及时获得响应 |

## 设计特点

- 🎨 **统一风格**：所有页面采用一致的卡片式布局设计
- 🌈 **渐变背景**：每个错误页面使用独特的渐变配色方案
- ✨ **动画效果**：包含淡入和脉冲动画，提升用户体验
- 📱 **响应式设计**：完美适配各种屏幕尺寸
- 🔗 **返回首页**：每个页面都包含返回首页的按钮

## 使用方法

1. 将需要的 HTML 文件复制到您的 Web 服务器错误页面目录
2. 配置 Web 服务器（Nginx、Apache 等）使用对应的错误页面
3. 根据需要自定义页面内容或样式

## 浏览器支持

- Chrome / Edge / Firefox / Safari 等现代浏览器
- IE11+（部分动画效果可能不支持）
